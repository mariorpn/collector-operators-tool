Instruções

Para coletar os dados, execute os seguintes comandos `oc` no seu terminal (logado no cluster desejado) e cole a saída JSON completa nos campos correspondentes abaixo.

oc get subscription.operators.coreos.com --all-namespaces -o json
oc get clusterserviceversion.operators.coreos.com --all-namespaces -o json
oc get clusterversion version -o json
